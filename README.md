## üöÄ Getting Started

### Prerequisites
- Linux operating system (or WSL on Windows)
- GCC compiler
- Basic knowledge of C programming

### Compilation
Most C programs can be compiled using:
```bash
gcc filename.c -o output
./output
```

For programs using threads:
```bash
gcc filename.c -pthread -o output
./output
```
